== Description ==

One Click Read More is a free plugin which allows you to replace "Continue reading" by "read more" tag.

Check [Demo and Features](https://webdev101.xyz/) .

###Example  
Continue Reading to Read more [Best Wordpress Hosting Reddit](https://www.reddit.com/r/webdev101/comments/r9llyb/best_wordpress_hosting_recommendations_2022/) .
Continue Reading to Read more [Best wordpress Hosting Reddit 2024](https://www.reddit.com/r/Webhosts/comments/180f7gn/what_web_hosting_is_best_for_wordpress_in_2024/) .



== Installation ==

1. Upload the 'one-click-read-more' folder to the '/wp-content/plugins/' directory.
2. Activate the "One Click Read More" list plugin through the 'Plugins' menu in WordPress.
